Configuration
=============

Loopa can be configured using environment variables or command-line arguments:

- `LOOPA_HOST`: Target host URL
- `LOOPA_USERS`: Number of users
- `LOOPA_RUN_TIME`: Test duration

Example:

    export LOOPA_HOST="http://example.com"
    export LOOPA_USERS=10
    export LOOPA_RUN_TIME="5m"
