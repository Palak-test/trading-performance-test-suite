# loopa

Loopa is a scalable, distributed load testing tool for web applications and services. It allows you to define user behavior, simulate traffic, and measure performance under various conditions.

## Features
- Easy scripting of user scenarios
- Distributed and scalable load generation
- Real-time web UI for monitoring
- Extensible with custom shapes and events
